# Reflections in the Rain 🌧️

A visually soothing **Random Quote Generator** with a rain animation background, inspired by reflective and introspective thoughts.

---

## Live Demo

Check out the live project here: [Reflections in the Rain](https://Ahmad-Ayman27.github.io/RandomQuoteGenerator1/)

---

## Features

- 💧 Rain animation over a scenic background  
- 📝 Displays random quotes with author attribution  
- 🎨 Smooth fade-in effect when showing a new quote  
- ✨ Stylish card layout with hover effects  
- 🌱 Calming and reflective user experience  

---

## Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling, animations, and hover effects  
- **JavaScript** – Random quote logic and interactive rain animation
